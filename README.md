# Txtar

Ruby implementation of [txtar](https://github.com/golang/tools/tree/master/txtar) Go package.

## Install

Add this line to your application's Gemfile:

```ruby
gem 'txtar'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install txtar
